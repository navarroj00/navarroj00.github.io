Bink2MP4

Bink2MP4 converts Bink Video (extension .bik) files which do not have audio into MP4 (extension .mp4) files.

You will need to have FFmpeg and The RAD Video Tools installed in order to properly utilize this script.

Bink2MP4 For Audio

Bink2MP4 For Audio converts Bink Video (extension .bik) files which have audio into MP4 (extension .mp4) files.

You will need to have FFmpeg and The RAD Video Tools installed in order to properly utilize this script.

DDS2PNG

DDS2PNG converts DirectDraw Surface (extension .dds) files into Portable Network Graphics (extension .png) files.

You will need to have Texconv installed in order to properly utilize this script.
