Projects


Bagels

This is a Python implementation of the deduction game Bagels which was required of me for my Introduction to Computer Science course.

Bink2MP4

Bink2MP4 converts Bink Video (extension .bik) files which do not have audio into MP4 (extension .mp4) files. You will need to have both FFmpeg and The RAD Video Tools installed in order to use this script.

Bink2MP4 For Audio

Bink2MP4 For Audio converts Bink Video (extension .bik) files which have audio into MP4 (extension .mp4) files. You will need to have both FFmpeg and The RAD Video Tools installed in order to use this script.

Carleton Fish Club Website

This is the official website for the now-defunct Carleton Fish Club at Carleton College. I was the president of this club and developer of its website.

DDS2PNG

DDS2PNG converts DirectDraw Surface (extension .dds) files into Portable Network Graphics (extension .png) files. You will need to have Texconv installed in order to use this script.

Earthquake Data Visualization

This is a Python implementation of a visualization based upon a data set of earthquakes which was required of me for my Introduction to Computer Science course.

Vigenere Cipher

This is a Python implementation of the Vigenère cipher which was required of me for my Introduction to Computer Science course.
