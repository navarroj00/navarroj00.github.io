Secret Hitler Probability Tables

Secret Hitler is a social deduction game in which two parties—the liberals and the fascists—are set to competitively enact liberal and fascist policies, respectively, or complete a secondary objective directly involving Hitler. The calculations included in this resource pertain to the probability of being assigned a particular role (i.e., Liberal, Fascist, or Hitler) and the probability of a particular policy card draw set occurring.

Secret Hitler rests on a hypergeometric distribution of probability for its policy card draw probabilities. This describes the probability of successes (random draws for which the object drawn has a specified feature) in draws, without replacement, from a finite population of size that contains exactly objects with that feature, wherein each draw is either a success or a failure.

Warframe Mobile API Endpoints

An application programming interface (API) is a computing interface which defines interactions between multiple software intermediaries. Basically, it is the part of a server that receives requests and sends responses.

I recovered these mobile API endpoints by capturing the traffic from the Warframe mobile application while it was running on my mobile device.
